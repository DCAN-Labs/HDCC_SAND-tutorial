Introduction
============

The size of structural volumes is potentially meaningful with respect to
brain function. Therefore, one fundamental question underlying ABCD and
HBCD studies are the developmental trajectories of brain regions.
BIBSnet provides volumetric outputs for the HBCD dataset. Here, we will
load BIBSnet data and measure the trajectory of volumetric data across
development. We will leverage ipython jupyter notebooks to perform this
analysis.

Module Objectives
=================

1. Gain familiarity with ipython jupyter notebooks

2. Explore volumetric outputs from BIBSnet

3. Develop longitudinal models to measure volume development from HBCD
   data
